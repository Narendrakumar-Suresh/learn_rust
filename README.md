# Learn Rust

A repository for learning Rust programming language basics and concepts.

## Topics Covered

- [ ] Variables and Data Types
- [ ] Functions and Control Flow
- [ ] Ownership and Borrowing
- [ ] Structs and Enums
- [ ] Pattern Matching
- [ ] Error Handling
- [ ] Collections
- [ ] Modules and Crates

## Getting Started

1. Install Rust from [rustup.rs](https://rustup.rs/)
2. Clone this repository
3. Run `cargo build` to compile projects
4. Run `cargo run` to execute examples
